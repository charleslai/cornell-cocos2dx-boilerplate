cornell-cocos2dx-boilerplate

Boilerplate for CS 4152 projects in Cocos2d-x at Cornell

Prerequesites

You must have the Cornell version of Cocos2d-x installed. Typing cocos in the command line should result in some kind of response. Please install the game engine to run the generator script.

Files

  • .gitignore - Custom .gitignore file
  • .travis.yml - Modular Travis CI YAML configuration
  • make_game.sh - Generator/Installation script
  • LICENSE.md - General MIT License that can be replaced
  • README.md - You're reading it

Generator Usage

./make_game.sh -c CompanyName -g GameName
